পরিচিতি

ভাষাLang হলো একটা toy programming language যেখানে সব keyword বাংলায়। এটা JavaScript-এর উপর চলে, তাই সব JS expressions কাজ করে।

Program Structure

প্রতিটা program শুরু করো দিয়ে শুরু হয় এবং শেষ করো দিয়ে শেষ হয়।

example.basha
শুরু করো

  // তোমার code এখানে

শেষ করো

Variables

ধরো দিয়ে variable declare করো।

vars.basha
শুরু করো
  ধরো naam = "রাহেলা"
  ধরো boyos = 20
  ধরো student = সত্যি
  বলো(naam)
  বলো(boyos)
শেষ করো

Print / Output

বলো() দিয়ে কিছু print করো — এটা console.log()-এর মতো।

print.basha
শুরু করো
  বলো("হ্যালো দুনিয়া!")
  বলো(1 + 2 + 3)
শেষ করো

Conditionals

যদি / নাহলে দিয়ে condition check করো।

if.basha
শুরু করো
  ধরো mark = 75
  যদি (mark >= 80) {
    বলো("A পেয়েছো!")
  } নাহলে {
    বলো("আরো পড়তে হবে।")
  }
শেষ করো

Loops

যতক্ষণ হলো while loop। থামো দিয়ে break করো।

loop.basha
শুরু করো
  ধরো i = 1
  যতক্ষণ (i <= 5) {
    বলো(i)
    i = i + 1
  }
শেষ করো

Functions

কাজ দিয়ে function declare করো। ফেরত দাও দিয়ে value return করো।

fn.basha
শুরু করো
  কাজ jog(a, b) {
    ফেরত দাও a + b
  }
  বলো(jog(10, 20))
শেষ করো
